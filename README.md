# Дашборд по вакцинации EU/EEA — Power BI

Интерактивный дашборд на основе открытых данных ECDC о вакцинации от COVID-19
в странах Евросоюза и ЕЭЗ.

## Данные
- Источник: ECDC (European Centre for Disease Prevention and Control)
- Охват: 30 стран, 2020–2023, 813 000+ строк

## Инструменты
- Power BI Desktop
- DAX (базовые вычисления), слайсеры, фильтры

## Что показывает дашборд
- **1,5 млрд** полученных доз вакцин суммарно
- **933 млн** первых доз
- **91 млн** экспортированных доз
- Разбивка по странам (лидер — Франция, 260+ млн доз) и возрастным группам
- Фильтры по периоду, стране и возрасту
